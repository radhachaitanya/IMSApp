Immigration Management System SetUp

Microsoft tool for Immigration management is being setup, which includes D365 solution components, Portal Components, supporting Azure components and additional master data. On successful deployments IMS system needs post configuration steps to be perfomred to complete the setup. It is advised that user may run this on vanila instance for best use. Later during uninstall of the solution may or may not remove all the components deployed during setup, which may need manual process to bring back the system.
